# TianShu / 天枢 - 项目计划文档

## 项目概述

TianShu（天枢）是一个综合性的IoT开发管理平台，旨在为物联网设备提供统一的管理、数据处理、API代理和AI Agent配置服务。

> 天枢，北斗七星之首，自古代表枢纽中心，寓意本平台作为物联网世界的核心枢纽。

### 核心理念
- **天枢**：作为设备、服务、AI的连接中心
- **可扩展**：模块化设计，支持快速扩展新设备和服务
- **开发友好**：为客户端应用提供便捷的API接口

---

## 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 后端框架 | Flask | 轻量灵活，生态丰富 |
| 数据库 | SQLite/PostgreSQL | 开发用SQLite，生产可切换PostgreSQL |
| ORM | SQLAlchemy | 数据库抽象层 |
| 缓存 | Redis (可选) | 数据缓存与消息队列 |
| MQTT | Mosquitto/EMQX | 物联网消息协议 (后期集成) |
| 任务队列 | Celery (可选) | 异步任务处理 |
| API文档 | Flask-RESTX/Swagger | 自动生成API文档 |

## API设计规范

### 基础URL
```
/api/v1/
```

### 响应格式
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

### 错误码
| 代码 | 说明 |
|------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 认证失败 |
| 1003 | 权限不足 |
| 2001 | 设备不存在 |
| 2002 | 设备离线 |
| 3001 | API调用失败 |
| 5000 | 服务器内部错误 |
