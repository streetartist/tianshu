# 天枢 TianShu

> 让智能设备真正智能起来

---

## 什么是天枢？

**天枢**是一个智能设备管理平台，就像智能设备的"大脑中枢"。

它能帮助您：
- 统一管理所有智能设备
- 让设备获得AI能力，变得更聪明
- 让设备能够以统一的方式使用api服务
- 查看设备状态和历史数据
- 远程配置和控制设备

---

## 能解决什么问题？

### 设备太多，管理混乱？
天枢提供统一的管理界面，所有设备一目了然。

### 想让设备更智能？
天枢可以为每个设备配置专属AI助手，让它能理解您的需求、主动提供服务。

### 需要接入天气、新闻等服务？
天枢内置API网关，轻松对接各类互联网服务，让设备获取实时信息。

### 想查看历史数据？
天枢自动记录设备上报的所有数据，随时查询、分析。
