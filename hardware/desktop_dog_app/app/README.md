# DesktopDog

由天枢IoT Studio生成的Flutter项目

## 初始化项目

```bash
flutter create . --org com.example --project-name desktopdog
flutter pub get
```

## 运行

```bash
flutter run
```

## 构建

```bash
flutter build apk      # Android
flutter build ios      # iOS
flutter build web      # Web
flutter build windows  # Windows
```
